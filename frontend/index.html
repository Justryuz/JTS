How It Works

Every prompt passes through TrustGuard before reaching your AI model.

01

Register & Get API Key

Create an account, register your domain, and generate a unique API Key. Stored as SHA-256 hash — shown once only.

02

Route Prompts Through Shield

Before sending any user prompt to your AI model, POST it to /api/v1/shield with your API Key.

POST /api/v1/shield

03

Act on the Response

Only forward the prompt to your AI model if TrustGuard returns ALLOWED. Block everything else.

BLOCKED → reject request
